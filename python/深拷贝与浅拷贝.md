# 浅拷贝

**概念** ：拷贝了最外围的对象本身，内部的元素都只是拷贝了一个引用而已。也就是，把对象复制一遍，但是该对象中引用的其他对象并不复制。

用一段代码就很容易理解
```
import copy
a = [1, 2, 3, [4]]
b = copy.copy(a)	# 浅拷贝

# 这里对a对象进行修改，看看a和b的不同
a[0] = 2
a[3][0] = 5
print('a-> {}'.format(a))
print('b-> {}'.format(b))
print('-'*20)		# 分割输出

# 这里用到id()这个方法，返回对象的唯一身份标识， is 判断语句就是判断这个方法的返回值
print('a-> {}'.format(id(a)))
print('b-> {}'.format(id(b)))
print('a[3]-> {}'.format(id(a[3])))
print('b[3]-> {}'.format(id(b[3])))
```

**输出**
```
a-> [2, 2, 3, [5]]
b-> [1, 2, 3, [5]]
--------------------
a-> 140437245149248
b-> 140437245149376
a[3]-> 140437245157632
b[3]-> 140437245157632
```
  从输出结果可以看出b复制a创建了一个新的对象，但对于a里的子对象[4], 并不是重新创建的，而是同一个对象，是直接引用a的子对象
